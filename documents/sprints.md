
## **MVP Sprint Breakdown (12 Sprints)**

---

### **Sprint 1 – Project Kickoff & Architecture**

**Goals:**

* Set up dev environment, repository, CI/CD.
* Define architecture for frontend, backend, and AI layer.
* Prepare UI wireframes.

**Deliverables Checklist:**
✅ Repo setup (GitHub/GitLab)
✅ React Expo + TypeScript scaffold
✅ Backend boilerplate (Node.js/NestJS or FastAPI)
✅ Firebase/Auth integration setup
✅ AI model access (OpenAI API key + LangChain)
✅ Wireframes approved (chat, onboarding, mood tracking)
✅ DevOps (CI/CD pipeline, staging environment)

---

### **Sprint 2 – Onboarding & Authentication**

**Goals:**

* Create user onboarding flow and authentication system.

**Deliverables Checklist:**
✅ Splash screen & app branding
✅ Sign-up/sign-in with email & phone
✅ Firebase Auth / JWT system
✅ Basic user profile creation
✅ Initial wellness questionnaire (baseline mental health)

---

### **Sprint 3 – Core AI Chat MVP**

**Goals:**

* Implement the AI mental health chat feature.

**Deliverables Checklist:**
✅ Chat UI (Gifted Chat or custom RN component)
✅ Connect AI backend to chat
✅ Session persistence (store conversation history in DB)
✅ Crisis keyword detection & flagging
✅ Basic prompt templates for empathetic responses

---

### **Sprint 4 – AI Chat Enhancements**

**Goals:**

* Improve AI accuracy, empathy, and personalization.

**Deliverables Checklist:**
✅ User tone preference in AI prompts (formal, friendly, neutral)
✅ Add typing indicators & message timestamps
✅ Sentiment analysis API integration
✅ AI-generated coping strategy suggestions
✅ Ability to bookmark chat snippets

---

### **Sprint 5 – Mood Tracking (Phase 1)**

**Goals:**

* Enable daily mood check-ins.

**Deliverables Checklist:**
✅ Mood selection (emoji scale)
✅ Quick notes/journaling input
✅ Store mood data in DB
✅ API for retrieving user’s mood history

---

### **Sprint 6 – Mood Tracking (Phase 2)**

**Goals:**

* Add analytics and AI-generated wellness reports.

**Deliverables Checklist:**
✅ Mood history chart (weekly/monthly)
✅ AI-generated weekly wellness summary
✅ Mood pattern notifications ("You’ve been stressed 3 days in a row")
✅ Offline mood logging with sync

---

### **Sprint 7 – Therapist Booking (Phase 1)**

**Goals:**

* Implement therapist discovery and booking system.

**Deliverables Checklist:**
✅ Therapist profiles (name, credentials, ratings)
✅ Search & filter for available therapists
✅ Appointment booking calendar
✅ Payment gateway (Stripe/Paystack) for session purchase

---

### **Sprint 8 – Therapist Booking (Phase 2)**

**Goals:**

* Integrate video call system & notifications.

**Deliverables Checklist:**
✅ Twilio/Agora video integration
✅ In-app session reminders (push notifications)
✅ Booking confirmation screens
✅ Therapist-side interface for accepting bookings

---

### **Sprint 9 – Self-Care Library**

**Goals:**

* Provide AI-personalized wellness content.

**Deliverables Checklist:**
✅ Resource library (articles, meditations, exercises)
✅ AI recommendation engine for resources
✅ Search & category filters
✅ Save-to-favorites feature

---

### **Sprint 10 – Security & Compliance**

**Goals:**

* Harden security and meet compliance standards.

**Deliverables Checklist:**
✅ End-to-end chat encryption
✅ GDPR/HIPAA compliance documentation
✅ Data anonymization routines
✅ Security penetration testing

---

### **Sprint 11 – Beta Testing & Feedback**

**Goals:**

* Launch closed beta, gather user feedback.

**Deliverables Checklist:**
✅ Deploy beta app to TestFlight & Google Play Internal Testing
✅ In-app feedback form
✅ Monitor analytics (Mixpanel/Firebase)
✅ AI model fine-tuning based on early chats

---

### **Sprint 12 – Launch Prep & Public Release**

**Goals:**

* Optimize, polish, and launch.

**Deliverables Checklist:**
✅ App store listings (copy, screenshots, keywords)
✅ Final bug fixes & performance optimization
✅ Marketing assets prepared (landing page, promo videos)
✅ Public launch on iOS & Android
